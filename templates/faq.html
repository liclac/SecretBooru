FAQ

Q: I get an error about a missing JPEG decoder when trying to import images!

You forgot to install libjpeg before installing PIL. Install libjpeg-dev/libjpeg-devel, then reinstall PIL with pip install -I PIL (the -I flag makes it ignore the installed package and reinstall it).

The details on how to install libjpeg varies by platform, but this is a quite common error, and there is plenty of documentation on the internet.

Q: I forgot my password!! D:

Well, crap. You're not getting anything out of SecretBooru without a password.

Every image is encrypted with an unique key, which is stored in a database, which is encrypted with your password. If you don't remember your password, you can't decrypt the keys needed to decrypt the images.
